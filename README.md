# أثير | منصة قرآنية متكاملة 🌿

أثير هي منصة قرآنية متكاملة، صُممت لتكون رفيقك الهادئ في تلاوة وحفظ القرآن الكريم. تجمع المنصة بين جمال التصميم وقوة الأداء، لتقدم لك تجربة روحانية عصرية في بيئة هادئة ومريحة للعين. 🌷

أثير ليست مجرد موقع، بل هي ملاذ رقمي يجمعك بكتاب الله بأسلوب يملأ قلبك بالسكينة، مع أدوات ذكية تساعدك على الحفظ والتدبر والاستماع. ❤️

---

##  الميزات الرئيسية

- ** أصواتٌ تخشع لها القلوب:** أكثر من 15 قارئاً من أعذب الأصوات، يمكنك الاختيار بينهم بسهولة، مع تشغيل تلقائي للسورة التالية أو تكرار السورة الحالية.

- ** مصحف مريح للعين:** عرض النص القرآني بالرسم العثماني الأصيل، مع وضع التسميع الذاتي لإخفاء الآيات واختبار حفظك بنقرة واحدة، والإشارات المرجعية لحفظ الآيات التي تحتاج لمراجعتها لاحقاً.

- ** التفسير الميسر:** تفسير معتمد من مجمع الملك فهد بالمدينة المنورة، يظهر فوراً عند النقر على أي آية.

- ** بحث ذكي:** ابحث عن السور باسمها أو رقمها بسرعة وسهولة.

- ** تصميم متكامل:** 6 ثيمات لونية (أخضر، وردي، بيج، أزرق، رمادي، بنفسجي) مع الوضع الليلي المدعوم لجميع الثيمات، وتصميم متجاوب يعمل على جميع الأجهزة.

- ** تطبيق تقدمي (PWA):** قابل للتثبيت على الهواتف والأجهزة اللوحية، ويعمل دون اتصال بالإنترنت بعد التحميل الأول.

- ** اختصارات لوحة المفاتيح:** تحكم كامل بتجربتك باستخدام لوحة المفاتيح (تشغيل/إيقاف، تقديم/تأخير، رفع/خفض الصوت، تفعيل وضع التسميع، إغلاق النوافذ).

---

##  التقنيات المستخدمة

- **HTML5** – هيكل التطبيق.
- **CSS3** – التصميم والثيمات والاستجابة لجميع الشاشات.
- **JavaScript (ES6)** – منطق التطبيق والتفاعلات.
- **IndexedDB** – تخزين البيانات محلياً للاستخدام دون اتصال.
- **Service Worker** – دعم العمل دون اتصال (PWA).
- **Web App Manifest** – تثبيت التطبيق على الأجهزة.

##  التثبيت والتشغيل محلياً

### المتطلبات الأساسية
- متصفح حديث (Chrome, Firefox, Edge, Safari)
- اتصال بالإنترنت (للتحميل الأول فقط)

### الخطوات

1. **استنساخ المستودع**
   ```
   git clone https://github.com/quran-dev/Atheer.git
   cd Atheer
   ```

2. **تشغيل التطبيق محلياً**
   - الطريقة الأولى (موصى بها): استخدام Live Server في VS Code
   - الطريقة الثانية: استخدام Python
     ```
     python -m http.server 8000
     ```
   - الطريقة الثالثة: فتح ملف `index.html` مباشرة (قد لا تعمل بعض الميزات)

3. **افتح المتصفح على الرابط:** `http://localhost:8000`

> **ملاحظة:** في الزيارة الأولى، سيتم تحميل البيانات من API وقد يستغرق ذلك بضع ثوانٍ. بعد التحميل، سيتم تخزين البيانات محلياً لاستخدامها دون اتصال. 🌷

## 🤝 كيفية المساهمة

نرحب بمساهماتكم! إليك كيفية المشاركة في تطوير أثير:

1. انشئ Fork للمشروع.
2. انشئ فرعاً جديداً للميزة أو الإصلاح.
